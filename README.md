Sign & Login Page

A modern, responsive, and reusable authentication UI containing both Sign Up and Login pages.
Built using HTML, CSS, and JavaScript, this template can be integrated into any website or web application.
Features

✔️ Fully responsive design

✔️ Clean & modern UI

✔️ Sign Up & Login forms

✔️ Smooth transitions

✔️ Basic form validation

✔️ Easy to connect with any backend

How It Works

The Login form handles authentication input.

The Sign Up form allows new user registration.

JavaScript performs:

Email validation

Empty field checks

Password length checks
